# Affan's Portfolio

A modern, responsive portfolio website built with React and featuring a premium dark theme with glassmorphism design and neon purple accents.

## Tech Stack

- **React 19** — UI framework
- **Vite 5** — Build tool & dev server
- **Tailwind CSS 3** — Utility-first styling
- **Framer Motion** — Animations & transitions
- **React Icons** — Icon library
- **React Router DOM** — Client-side routing

## Features

- Responsive design (mobile, tablet, desktop)
- Glassmorphism UI with neon glow effects
- Smooth scroll navigation
- Animated sections with scroll-triggered reveals
- Project showcase with category filtering
- Downloadable resume
- Contact section with social links

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This project is configured for deployment on [Vercel](https://vercel.com). Security headers are pre-configured in `vercel.json`.
